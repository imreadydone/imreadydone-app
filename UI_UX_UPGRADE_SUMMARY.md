**"Vibrant Neo-Brutalist Korean Todo"**

- **차별점**: 강렬한 그라디언트, 대담한 타이포그래피, 유쾌한 애니메이션
- **목적**: 할 일 관리를 즐겁고 활기찬 경험으로 만들기

## ✨ 주요 변경사항

### 1. **전면적인 디자인 시스템 재구축**

#### 타이포그래피
- **메인 폰트**: Pretendard Variable (한글 최적화, 가변 폰트)
- **로딩**: CDN (jsdelivr)
- **효과**: 부드러운 안티앨리어싱

#### 색상 팔레트
```
메인 그라디언트: 골드(#FFD700) → 오렌지(#FF8C00)
악센트 색상:
  - 핑크: #FF6B9D
  - 시안: #00D4FF
  - 라임: #CCFF00
베이스 색상:
  - 다크 네이비: #0A0E27
  - 네이비 라이트: #1A1F3A
  - 네이비 라이터: #2A2F4A
```

#### Brutal 그림자 시스템
```css
기본: 6px 6px 0px rgba(255, 215, 0, 0.4)
호버: 8px 8px 0px rgba(255, 215, 0, 0.6)
```

#### 애니메이션
1. **slideInUp**: 요소 등장 (0.4s ease-out)
2. **scaleIn**: 스케일 애니메이션 (0.3s ease-out)
3. **shake**: 오류 표시 (0.4s ease-in-out)
4. **pulse-glow**: 글로우 효과
5. **gradient-shift**: 그라디언트 이동

### 2. **메인 페이지 UI 고도화**

#### 헤더
- 대형 그라디언트 타이틀 (56px → 60px)
- 서브 타이틀 추가: "할 일을 완료하자! 🚀"
- 뷰 모드 토글 버튼 강화 (골드 하이라이트)
- 로그아웃 버튼 (핑크 악센트)

#### 할 일 추가 폼
- 4px 두께 보더
- 폰트 크기 및 굵기 증가
- Brutal 그림자 효과 (추가 버튼)
- 호버/포커스 효과 강화

#### TodoCard 컴포넌트
**비주얼**:
- 4px 두께 테두리 (호버 시 핑크로 변경)
- Brutal 그림자 + 호버 리프트 효과
- 스케일 인 애니메이션

**배지 강화**:
- D-day: 색상별 분류 (레드, 오렌지, 시안)
- 서브태스크: 핑크 배지
- 카테고리: 라임 배지

**상호작용**:
- 상태 토글 버튼 (스케일 125%)
- 상태 변경 버튼 (칸반 전용)
- 삭제 버튼 (호버 시 핑크)

**아코디언 상세 정보**:
- 시안 구분선
- AI 분석 (핑크 레이블)
- 서브태스크 (라임/핑크 레이블)
- 태그 (시안 배지)
- 에이전트 (라임 텍스트)

#### 칸반 보드
- 컬럼 헤더 강화 (bold, 20px)
- 카운트 배지 색상 차별화:
  - 대기: 골드
  - 진행 중: 시안
  - 완료: 라임
- 빈 상태 메시지 개선

### 3. **인증 폼 UI 개선**

#### 레이아웃
- 대형 그라디언트 타이틀 (60px)
- 서브 헤딩 (이모지 포함)
- 중앙 정렬 레이아웃

#### 폼 요소
- 레이블 추가 (색상별: 시안/핑크)
- 입력 필드 강화 (4px 보더, 세미볼드)
- 제출 버튼 Brutal 스타일 (그림자 + 호버 스케일)
- 로딩 스피너 (회전 애니메이션)

#### 효과
- 오류 메시지: Shake 애니메이션
- 토글 버튼: 시안 → 핑크
- 하단 데코레이션 점 (펄스 애니메이션, 지연)

### 4. **글로벌 스타일**

#### 커스텀 요소
- 스크롤바: 그라디언트 (골드 → 오렌지)
- 선택 텍스트: 핑크 배경
- 포커스 링: 골드 글로우

#### 유틸리티 클래스
```css
.gradient-text        - 그라디언트 텍스트
.brutal-shadow        - Brutal 그림자 + 호버 효과
.animate-slide-up     - 슬라이드 업
.animate-scale-in     - 스케일 인
.animate-shake        - 흔들림
```

## 📊 성능 고려사항

### 최적화
- **폰트 로딩**: CDN 프리커넥트
- **애니메이션**: CSS 전환 (GPU 가속)
- **그라디언트**: CSS gradients (이미지 대신)
- **색상**: CSS 변수 (재사용성)

### 번들 크기
- **Pretendard**: ~100KB (동적 서브셋)
- **CSS**: ~4KB (압축)
- **JS**: 변화 없음
