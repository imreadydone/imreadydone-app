# I Am Ready Done - 디자인 시스템

## 📋 개요

I Am Ready Done의 디자인 시스템은 일관성 있고 접근 가능한 사용자 경험을 제공하기 위한 가이드라인입니다.

## 🎨 핵심 원칙

### 1. **다크 퍼스트 (Dark-First)**
- 주 사용 시간대가 야간임을 고려한 다크 테마 우선 설계
- 눈의 피로를 최소화하는 부드러운 명암비
- 고대비 모드 지원으로 접근성 강화

### 2. **이모지 중심 아이콘 시스템**
- 별도의 아이콘 라이브러리 없이 유니버설한 이모지 활용
- 직관적이고 친근한 시각적 언어
- 다국어 환경에서도 일관된 의미 전달

### 3. **마이크로 인터랙션**
- 사용자 행동에 즉각적인 피드백
- 부드럽고 자연스러운 애니메이션
- 성능을 고려한 CSS 우선 애니메이션

### 4. **모바일 퍼스트 반응형**
- 터치 타겟 최소 44x44px (WCAG AAA 준수)
- 모바일에서 데스크톱으로 확장되는 설계
- 유연한 그리드 시스템

### 5. **접근성 우선**
- WCAG 2.1 AAA 레벨 준수
- 키보드 네비게이션 완전 지원
- 스크린 리더 최적화

## 🛠️ 기술 스택

- **프레임워크**: Next.js 16.1.6 (App Router)
- **UI 라이브러리**: React 19.2.3
- **스타일링**: Tailwind CSS 4.x
- **백엔드**: Firebase (Firestore, Auth, FCM)
- **언어**: TypeScript 5.x

## 🎯 디자인 토큰

디자인 토큰은 Tailwind CSS의 기본 설정을 확장하여 사용합니다.

```typescript
// 예시: 주요 컬러 토큰
--color-primary: #3b82f6;      // blue-600
--color-danger: #dc2626;       // red-600
--color-success: #16a34a;      // green-600
--color-warning: #ea580c;      // orange-600

// 배경 컬러
--bg-primary: #030712;         // gray-950
--bg-secondary: #1f2937;       // gray-800
--bg-tertiary: #111827;        // gray-900
```
