## 🎭 시각 효과 및 애니메이션

### 1. Glass Morphism (유리 효과)
```css
background: rgba(30, 43, 39, 0.6);
backdrop-filter: blur(16px);
border: 1px solid rgba(82, 217, 163, 0.1);
```
- 카드와 입력 필드에 적용
- 콘텐츠가 투과되며 깊이감 연출
- 모던한 UI 트렌드 반영

### 2. Glow Effects (발광 효과)
- 중요한 액션 버튼에 은은한 빛 효과
- 포커스 상태에서 그린 글로우 링
- 완료된 작업 수에 미묘한 펄스 효과

### 3. 살아있는 배경
- **Gradient Animation**: 15초 주기로 부드럽게 변화하는 그라데이션
- **Floating Orbs**: 3개의 블러 처리된 원이 천천히 떠다님
- **Noise Texture**: 미세한 노이즈 오버레이로 유기적 질감

### 4. 마이크로 인터랙션
| 요소 | 효과 |
|------|------|
| 버튼 호버 | 리프트 + 그림자 강화 |
| 버튼 클릭 | 중앙에서 퍼지는 ripple 효과 |
| 체크박스 호버 | 10% 확대 + 색상 변화 |
| 체크박스 체크 | 부드러운 체크마크 애니메이션 |
| 카드 호버 | 4px 상승 + 그림자 |
| 입력 포커스 | 그린 글로우 링 (3px) |

### 5. 페이지 로드 시퀀스
```css
.stagger-children
```
- 각 TODO 카드가 순차적으로 fade-in + slide-up
- 50ms 간격으로 계단식 등장
- 우아한 첫인상 연출

   - 현재는 다크 모드만 지원

2. **커스텀 테마**
   - 사용자가 색상 선택 (계절별 팔레트)
   - 가을(주황), 겨울(푸른), 봄(연두), 여름(그린)

3. **일러스트레이션**

   - 완료 시 꽃피는 애니메이션

4. **사운드**
   - 체크 시 부드러운 클릭음
   - 완료 시 새소리 or 종소리
   - 사용자 선택 가능 (on/off)

5. **완료 축하**
   - Confetti 애니메이션
   - 격려 메시지 팝업

6. **성장 시각화**
   - 진행률에 따라 식물이 자라는 일러스트
   - 주간/월간 완료 통계 그래프

7. **드래그 앤 드롭**
   - 칸반 보드에서 카드 드래그 이동
   - React Beautiful DnD or dnd-kit 활용

8. **다크 패턴 제거**
   - 알림 배너를 덜 공격적으로 (작게, 숨기기 가능)

9. **성능 최적화**
   - 폰트 자체 호스팅 (FOUT 방지)
   - 이미지 최적화 (WebP, AVIF)
   - 코드 스플리팅

10. **PWA 기능 강화**
    - 오프라인 모드
    - 앱 아이콘 커스터마이징
    - 위젯 지원 (Android)
